# **App Name**: AlumniConnect

## Core Features:

- Verified Alumni Directory: Browse a directory of verified alumni, with profiles showcasing their experience, skills, and career paths.
- Opportunity Feed: Alumni can post internships, project openings, referrals, and announcements in a social-feed style.
- Direct Messaging: Enable one-to-one chat between students and alumni for follow-ups and guidance.
- Smart Matchmaking: Match students with alumni mentors based on domain, skills, interests, and career goals using a generative AI tool that reasons over user profiles.
- Student Profiles: Allow students to create detailed profiles highlighting their academic background, interests, and career aspirations.
- Query and Guidance System: Students can post questions and request guidance on academics, careers, and skill development.
- Role-Based Access Control: Implement role-based access to manage student and alumni privileges.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean and modern feel.
- Accent color: Teal (#38B2AC) for highlighting important actions and elements.
- Headline font: 'Space Grotesk' sans-serif for a modern and techy feel; body font: 'Inter' sans-serif.
- Use consistent and professional icons to represent different areas of guidance (academics, career, etc.).
- Clean and intuitive layout for easy navigation and user experience.
- Subtle transitions and animations to enhance user engagement.
